This script finds which numbers between 1 and 10 are missing from a given list.

It receives a comma-separated list of numbers via the command line and returns the missing values within the range.

### Usage

Run the script with:

```bash
node index.js <comma-separated list of numbers>
```

Example:
```bash
node index.js 10, 5, 1, 2, 4, 6, 8, 3, 9
// output = = 7
```
